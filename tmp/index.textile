h2. ブロックレベルの構造

<pre>
    セクション 1
      セクション 1.1
    セクション 2
</pre>
p. の様に、下位レベルのセクションは、上位レベルのセクションに含まれる

p. セクションのタイトルは、'*' -&gt; &lt;h2&gt; から始まり、' **** ' 以降は、UL 
  として出力される。
  &lt;h&gt; と &lt;ul&gt; の閾値は、#+OPTIONS: H:n で指定出来る。
  （デフォルトは、n=3)

h2. タイトル

p. 最初のテキストラインが、待った無しでタイトルになる。
  最初のヘッドラインまでの部分も、出力される。
<pre>
title

xxxx
* 1st headline
</pre>
p. の場合、
<pre>
<h1>title</h1>

xxxx
<h2>1st headline</h2>
</pre>
p. の様に変換。[1]

p. ヘッドラインから始まる場合は、最初のヘッドラインの内容がタイトルとなる。

h3. #+TITLE: xxxx

<pre>
#+TITLE: xxxx
</pre>
p. が指定されている場合は、xxxx がタイトルになる。


h3. Options: skip:t

<pre>
#+OPTIONS: skip:t
</pre>
p. が指定されていると、最初のセクション以前のテキストは捨てられる。
   タイトルには、1st headline が使われる。

p. この場合も、
<pre>
#+TITLE: xxxx
</pre>
p. で、タイトルを指定出来る。

h2. Section node

</pre>

p. COMMENT から始まるヘッドラインを持つ Section は、全体をコメントとして扱う。

h3. Paragraph node

p. ヘッドラインの次の行から始まり、セクションの最後か、１行以上の空行で終わる部分は、
   段落として扱う。段落中で、改行させたい場合には、行末に"\\"を置く。

<pre>
#+BEGIN_VERSE 〜 #+END_VERSE
</pre>
p. で囲われた部分は、改行が保存される。
    "verse example":./verse.org   "html":./verse.html 

<pre>
     <p class="verse">
     ...<br/>
     &nbsp;&nbsp;...<br/>
     </p>
</pre>

p. の様に、展開される様だ。

h4. #+ATTR_HTML

p. If you want to specify attributes for links, 
    you can do so using a special #+ATTR_HTML line 
    to define attributes 
    that will be added to the &lt;a&gt; or &lt;img&gt; tags. 

p. Here is an example that sets title and style attributes for a link: 
<pre>
  #+ATTR_HTML: title="The Org-mode homepage" style="color:red;"
   http://orgmode.org 
</pre>
